# zivsb.github.io

My personal homepage — a maximalist Web 1.0 / Y2K tribute, hand-built as a
single static page. Lives at <https://zivsb.github.io>.

## Contents

- `index.html` — the whole site (HTML + CSS + a little JS, no build step)
- `email.gif` — my email rendered as an animated image so it never appears as
  scrapable text anywhere in the source
- `Ziv_Weissman_CV.pdf` — CV, linked from the page

## Notes

- The visitor counter uses [counterapi.dev](https://counterapi.dev) so it works
  on static hosting.
- The top "connection established" bar only shows when the browser exposes a
  real bandwidth estimate (Network Information API).
- Best experienced at 800×600 with the blinds drawn. 🛰
